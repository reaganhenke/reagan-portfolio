Tensorflow Rock, Paper, Scissors (2024)

This Angular site uses tensorflow.js, a library for machine learning, to play Rock, Paper, Scissors. Video from the webcam is passed to a pre-trained image recognition model that detects a hand and returns 21 landmark keypoints. The computer can then play Rock, Paper, Scissors with varying levels of "intelligence".

The site is currently live at tensorflowrockpaperscissors.web.app. Code is available at RockPaperScissors Github.
